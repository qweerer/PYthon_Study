# 阅读-判断

- 原因在判断题里有排他性质（唯一性质）
- 数字不同一定false
- 每道题对应1-2句

> 💎 抓考点
> - 01.动作，如果动作太泛泛则考后面的宾语
> - 02.be动词后是考点
> - 03.宾语从句为重点
> - 04.数字，原因，比较
> - 05.不要管定位状语

==TRUE出题规则==
1. 换词
2. 换词性
3. 换句式
4. 文具体=提概括
5. 正话反说
==False出题规则==
1. 直接否定
2. 数字不同
3. 一正一负
4. 关系不同
5. 矛盾
==NG规则==
1. 不矛盾
2. 未提及
3. 混搭
4. 虚假比较
5. 范围缩小
