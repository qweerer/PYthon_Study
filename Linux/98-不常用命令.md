## 字体

```shell
# 本地字体目录
/usr/local/share/fonts

# 查看所有字体
fc-list
# 更新字体
fc-cache

# setfont设置终端字体
# setfont在x下没法使用，只能是在EGA/VGA字符终端下可用
# 查看所有字体样式
ls /usr/share/kbd/consolefonts
# 设置当前终端字体为指定名称的字体。
setfont font_name
setfont -C /dev/tty2 font_name
```
