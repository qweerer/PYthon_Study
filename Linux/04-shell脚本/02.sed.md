# 02.sed

```shell
# 匹配前后行加入文字
sed -i '/2222222222/a\3333333333' test.txt
sed -i '/2222222222/i\3333333333' test.txt
# 在某行(指具体行号)前或后加一行内容
sed -i 'N;4a\ddpdf' a.txt
sed -i 'N;4i\eepdf' a.txt

# 插入到某行
# 直接加到36行：行首（^表示行首）加四个空格（缩进）和参数：
sed -i '36s/^/ client_max_body_size 50m;/g' nginx.conf
sed -i '1s|^| client_max_body_size 50m;\n|' nginx.conf

```


## 删除
```shell

sed -i '1d' filename ### 删除第一行
sed -i '1,3d' filename ### 范围删除，删除1-3行
sed -i 'nd' filename ### 删除第n行
sed -i '$d' filename ### 删除最后一行
```
