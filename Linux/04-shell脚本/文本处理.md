
## column

```bash
# 示例
column -t -s ":" -N "ONE,TWO,THREE,FOUR,FIVE,SIX,SEVEN" -T "FIVE,SEVEN" -o "|" /etc/passwd

# 解析
column  -t # 转变为table
	    -s ":" # 设置分隔符
	    -o "|" # 边缘线
	    -N , --table-columns="user,N,uid,uid,describe,home,shell" # 设置行标题
	    -H , --table-hide="describe" # 隐藏describe行
	    -T , --table-truncate="describe,shell" # 缩短列
	    -J -n "pawwsd" # 输出一个名为"passwd"的json
```
