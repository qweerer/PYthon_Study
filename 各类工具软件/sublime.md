# sublime

## 快捷键

| 快捷键  | 作用                  |
| ------- | --------------------- |
| *C+S*+p | 快捷命令              |
| *C*+p   | 搜索文件，:80跳到80行 |
| *C*+R   | 搜索函数              |
| *C+S*+r | 搜索函数，在项目中    |
| *A* -   | 在操作的地方跳转      |
| *C*+d   | 选中所有选择的单词    |
