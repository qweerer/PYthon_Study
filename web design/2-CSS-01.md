## 在html文件中定义

```html
<!DOCTYPE html>
<html lang="en">
<head>
 <meta charset="UTF-8">
 <meta http-equiv="X-UA-Compatible" content="IE=edge">
 <meta name="viewport" content="width=device-width, initial-scale=1.0">
 <title>表格案例</title>
 <style>
	 p {
	 	color: red;
		font-size: 30px;
	 }
 </style>
</head>
```
