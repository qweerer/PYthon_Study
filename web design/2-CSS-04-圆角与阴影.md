## 盒子阴影
```css
.showbox:hover {
 	box-shadow: 0px 10px 10px -5px rgba(0, 0, 0, 0.3);
 }
```

> box-shadow: h-shadow, v-shadow, blur, spread, color, inset

| 值       | 描述                            |
| -------- | ------------------------------- |
| h-shadow | 必须,水平阴影位置,可以为负值    |
| v-shadow | 必须,垂直阴影位置,可以为负值    |
| blur     | 可选,模糊距离                   |
| spread   | 可选,影音尺寸                   |
| color    | 可选,阴影颜色                   |
| inset    | 可选,将外部阴影(outset)改为内部 |

## 文字阴影
