## 标签

| 功能                |              代码              |
| ------------------- |:------------------------------:|
| <b>加粗</b>         | `<strong></strong>`, `<b></b>` |
| <em>倾斜</em>       |     `<em></em>`, `<i></i>`     |
| <del>删除线</del>   |    `<del></del>`, `<s></s>`    |
| <ins>下划线</ins>   |    `<ins></ins>`, `<u></u>`    |
| <div>万能标签</div> | `<div></div>`, `<span></span>` |
