# 自己找的一些操作,补充性质

## 字符格式化

### 字符宽度

>用于对齐数字,文字.2d表示最少2个字符宽度

```python
print('%2d'%1)
print('%2d'%10)
print('%2d'%100)
```

```c
 1
10
100
```

## 随机

```py
import random
```

### 打乱列表

```py
random.shuffle(list1)
```

无返回值,直接打乱list1

### 随机生成布尔值嵌套列表

```py
[[True if random.randint(0, 1) == 1 else False for i in range(3)] for j in range(5)]
```
